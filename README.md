## Grid Layout App

**Grid Layout App** is a layout design made with flutter and other usable components.

## Installation

In your terminal, type these commands

```bash
#clone the repo
git clone https://github.com/sathakhussam/crwn-clothing
cd grid-layout-app-flutter

#Running the App
flutter run
```
